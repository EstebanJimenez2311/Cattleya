¿Qué es la estadística inferencial?
A diferencia de la estadística descriptiva (que describe lo que ya ocurrió), la estadística inferencial nos permite sacar conclusiones sobre una población completa a partir de una muestra, y cuantificar qué tan confiables son esas conclusiones. En este proyecto la usamos para responder preguntas como: ¿existe una relación estadísticamente significativa entre la región geográfica y el tipo de violencia reportado? ¿Hay diferencias reales entre años, o son solo variaciones del azar?

¿Qué es el Machine Learning y por qué lo aplicamos aquí?
El machine learning (aprendizaje automático) permite que los computadores aprendan patrones a partir de datos históricos para hacer predicciones o clasificaciones sobre casos nuevos. En este proyecto lo usamos para identificar qué factores (región, edad, tipo de relación, etc.) tienen mayor peso en predecir el tipo o la severidad de los casos reportados. Esto convierte el análisis de datos en una herramienta de anticipación y política pública.
